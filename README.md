# Booster - Boost Converter Loss Calculator

**Booster** is a simple, lightweight utility designed to help with the calculation of losses in boost switching regulators. 

### What it does:
* **CCM & DCM Analysis:** Provides a visual graph showing the transition between Continuous and Discontinuous Conduction Modes.
* **Loss Estimation:** Calculates basic switching and conduction losses to help you size your components.
* **Fast Iterations:** A quick way to check your design before moving to full SPICE simulations.

### Accuracy:
I developed this tool for my own needs and have verified the results against **SPICE simulations**. While it's not a replacement for full circuit simulation, it provides a reliable and close approximation for initial design steps.

### How to use:
1. Download the `booster.exe` from the [Releases](../../releases) section.
2. No installation needed—just run the file.
